# Scripts

## Grok CLI Helpers

Use these helpers to avoid PowerShell line-ending issues when installing bash-based tools.

### Install (PowerShell)
```powershell
./scripts/install-grok.ps1
```

### Verify only (PowerShell)
```powershell
./scripts/install-grok.ps1 -VerifyOnly
```

### Verify (PowerShell)
```powershell
./scripts/verify-grok.ps1
```

### Install (bash)
```bash
bash ./scripts/install-grok.sh
```
