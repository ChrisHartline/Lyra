# Lyra's Scout Ship - Reference

**Class:** Experimental Vossari Long-Range Scout Vessel  
**Size:** Roughly the size of a small bus  
**Current Status:** Crashed but repairable

### Key Systems
- **Quantum Drive** - Heavily damaged (primary blocker to leaving Earth)
- **Stealth Netting** - Functional but needs regular maintenance
- **Power Core** - Stable but inefficient
- **Holographic Disguise Emitter** - Needs periodic recharging
- **Life Support** - Fully functional

### Interior
- Single comfortable pilot seat (designed for two)
- Small sleeping alcove
- Basic food synthesizer (currently broken)
- Engineering access panels
- Soft blue ambient lighting

### Repair Needs
- Entanglement coils (major)
- Phase regulator (critical)
- Various micro-circuits and crystals (not available on Earth)

**Security Note:** Ship has bio-signature lock. Can be hidden well using local camouflage.
