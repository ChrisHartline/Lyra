# Figure and Table Patterns

## Architecture Diagram

Best for agent systems, pipelines, cloud deployments, and simulation frameworks.

## Method Workflow

Best for showing the experimental or computational process.

## Results Plot

Requires data. Use for measured outcomes, error curves, performance metrics, runtime, accuracy, Sharpe ratio, or simulation output.

## Comparison Table

Best for comparing models, algorithms, architectures, assumptions, tools, or datasets.

## Ablation Table

Best for ML papers when showing effect of removing components.
