# IEEE Formatting Notes

Use `IEEEtran` when producing IEEE-compatible LaTeX.

Default conference class:

```latex
\documentclass[conference]{IEEEtran}
```

Default journal class:

```latex
\documentclass[journal]{IEEEtran}
```

## Common Rules

- Use `\IEEEauthorblockN` and `\IEEEauthorblockA` for conference authors.
- Use numbered citations via BibTeX.
- Keep captions concise and descriptive.
- Use `\label{}` immediately after `\caption{}`.
- Avoid overly wide tables.
- Prefer vector figures when possible.
