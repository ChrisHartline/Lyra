# LaTeX Conventions

## Labels

Use prefixes:

- `sec:` for sections
- `fig:` for figures
- `tab:` for tables
- `eq:` for equations
- `alg:` for algorithms

## TODOs

Use LaTeX comments:

```latex
% TODO: Add evaluation details.
```

## Tables

Prefer simple tables with `booktabs` where allowed.

## Figures

Prefer PDF, SVG converted to PDF, EPS, or high-resolution PNG depending on venue.
